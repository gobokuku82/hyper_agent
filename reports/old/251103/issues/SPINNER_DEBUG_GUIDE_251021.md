# Spinner 문제 디버깅 가이드 (사용자용)

**목적**: 복합 질문 시 spinner가 작동하지 않는 원인을 정확히 파악하기
**소요 시간**: 5분
**난이도**: 매우 쉬움 (스크린샷만 찍으면 됨)

---

## 🎯 간단 요약

**지금 해야 할 일**: 복합 질문을 입력하고, F12 눌러서 화면 보고, 스크린샷 찍기

---

## 📋 Step 1: 준비 (10초)

1. **브라우저에서 프론트엔드 열기**
   - 주소: http://localhost:3000 (또는 현재 사용 중인 주소)

2. **F12 키 누르기**
   - 개발자 도구가 열립니다

3. **Console 탭 선택**
   - 아래쪽에 Console 탭이 보입니다
   - 클릭해서 활성화

---

## 📋 Step 2: 복합 질문 입력 (5초)

**입력할 질문**:
```
강남구 아파트 시세 확인하고 투자 분석해줘
```

또는:
```
전세 시세 조회하고 위험도 분석해줘
```

→ 전송 버튼 클릭

---

## 📋 Step 3: 화면 관찰 (30초)

### 3-1. Spinner가 보이는가?

**확인할 것**:
- "작업 실행 중" 카드가 보이는가?
- Settings 아이콘(⚙️ 톱니바퀴)이 보이는가?
- 톱니바퀴가 **회전하는가?** (이것이 핵심!)

**스크린샷 찍기**:
- Windows: `Win + Shift + S` → 영역 선택
- 또는 전체 화면: `Print Screen`

### 3-2. Console 로그 확인

**Console 탭에서 찾을 내용**:
```
[ChatInterface] Received WS message: execution_start
```

**스크린샷 찍기**:
- Console 전체를 캡처

---

## 📋 Step 4: Elements 탭 확인 (선택사항, 1분)

1. **Elements 탭 클릭**
   - Console 옆에 있음

2. **Ctrl + F 누르기**
   - 검색창이 나타남

3. **"작업 실행 중" 입력**
   - 자동으로 해당 요소를 찾아줌

4. **결과 확인**:
   - 찾았나? → "있음"
   - 못 찾았나? → "없음"

**스크린샷 찍기**:
- Elements 탭 전체를 캡처

---

## 📋 Step 5: 백엔드 로그 확인 (선택사항, 30초)

**PowerShell 또는 CMD에서 실행**:

```powershell
cd C:\kdy\Projects\holmesnyangz\beta_v001\backend
Get-Content logs\app.log -Tail 100 | Select-String "execution_start"
```

**또는 간단하게**:
```powershell
cd C:\kdy\Projects\holmesnyangz\beta_v001\backend
notepad logs\app.log
```
→ 메모장에서 `Ctrl + F` → "execution_start" 검색

**찾을 내용**:
```
[TeamSupervisor] Sent execution_start via WebSocket
```

**스크린샷 찍기**:
- 로그 내용 캡처

---

## 📊 결과 정리 (제공해 주세요)

### 필수 정보

**1. Spinner 보임 여부**:
- [ ] "작업 실행 중" 카드가 보임
- [ ] Settings 아이콘(톱니바퀴)이 보임

- [ ] `[ChatInterface] Received WS message: execution_start` 보임
- [ ] 안 보임

### 선택 정보

**3. Elements 탭**:
- [ ] "작업 실행 중" 요소 있음
- [ ] 없음

**4. 백엔드 로그**:
- [ ] `Sent execution_start` 있음
- [ ] 없음

## 💡 간단 요약

**사용자가 할 일**:

1. F12 누르기
2. 복합 질문 입력 ("강남구 아파트 시세 확인하고 투자 분석해줘")
3. 화면 보고 스크린샷 찍기:
   - 톱니바퀴가 회전하나? (예/아니오)
   - Console에 "execution_start" 보이나? (예/아니오)

→ 이 2가지만 알려주면 됩니다!

---

## 🎯 가장 쉬운 방법 (1분)

**이것만 해도 됩니다**:

1. 복합 질문 입력
2. "작업 실행 중" 카드 나타나면 스크린샷
3. F12 → Console 탭 스크린샷

→ 2장의 스크린샷만 주시면 원인 파악 가능합니다.

---

## ❓ 자주 묻는 질문

**Q1. F12가 안 눌러지는데요?**
- 우클릭 → 검사(Inspect) 클릭해도 됩니다

**Q2. Console 탭이 안 보이는데요?**
- 개발자 도구 상단에 Console, Elements, Network 등이 있습니다
- Console 클릭

**Q3. 로그가 너무 많아요**
- `Ctrl + L`로 화면 지우기
- 그 다음 복합 질문 입력
- 새로운 로그만 보이게 됩니다

**Q4. "execution_start"를 못 찾겠어요**
- Console에서 `Ctrl + F` 누르면 검색창 나옵니다
- "execution" 입력하면 하이라이트됩니다

**Q5. 스크린샷을 어디에 저장하나요?**
- `Win + Shift + S` 사용 시 클립보드에 복사됩니다
- 채팅에 `Ctrl + V`로 붙여넣기 하면 됩니다
